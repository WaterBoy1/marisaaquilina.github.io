Get Outdoors Day
Web App Case Study

Developed as a case study for camping and land-sharing platform startup Hipcamp, this web app displays user-submitted content to promote National Get Outdoors Day.

Launch

The Pledge feature allows users to enter their email address for a one-time email blast from the Hipcamp team reminding them to get outdoors a few days prior to the holiday. It also allows users to invite their friends to pledge on the web app.

The Exploration Mapper feature plots the locations of Hipcampers on Get Outdoors Day by retreiving the 20 most recent Instagram posts with the #GetOutdoorsDay tag and a set location. This feature invites users to see where others have gone as well as submit their own content.

Many elements of the web app's interface is currently hard-coded. However, I have designed the logic behind the full implementation of this web app.
